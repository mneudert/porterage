# Porterage

Checks, fetches and delivers configurable datasources.

## Setup

Add the library as a dependency to your `mix.exs` file:

```elixir
defp deps do
  [
    # ...
    {:porterage, "~> 0.1.0"},
    # ...
  ]
end
```
